Metatable of FLOX variables with units and description
Variable Units Description
doy.dayfract [unitless] Local time and date in decimal format, e.g 179 is the 00.00 o'clock on 179th day of the calendar year, noon is 179.5 and so on.
datetime [UTC] [date] UTC Time downloaded from GPS
SZA [degrees] Sun Zenith Angle, calculated from UTC time
Lat [degrees] Latitude downloaded from GPS
Lon [degrees] Longituted downloaded from GPS
Incoming at 750nm [W m-2nm-1sr-1] Incommin radiation at 750 nm
Reflected 760 [W m-2nm-1sr-1] Reflected radiation at 760 nm
Reflected 687 [W m-2nm-1sr-1] Reflected radiation at 687 nm
Reflectance 750 [unitless] Reflectance (unitless)
Reflectance 760 [unitless] Reflectance (unitless)
temp1 [C] Temperature readings inside box, for more info contact JB
temp2 [C] Temperature readings inside box, for more info contact JB
temp3 [C] Temperature readings inside box, for more info contact JB
temp4 [C] Temperature readings inside box, for more info contact JB
h1 [%] Relative Humidity
h2 [%] Relative Humidity
E_stability [%] Percentage difference between first and last PAR readings for measurement cycle from QE Pro. Shows how dynamic Ilumination conditions during measurement are
sat value L [unitless] Saturation value for reflected radiation from QE Pro
sat value E [unitless] Saturation value for incoming radiation at begining of measurement cycle from QE Pro
sat value E2 [unitless] Saturation value for incoming radiation at end of measurement cycle from QE Pro
SIF_A_ifld [mW m-2nm-1sr-1] Sun induced fluorescence values for 760 nm retrieved by iFLD
SIF_B_ifld [mW m-2nm-1sr-1] Sun induced fluorescence values for 687 nm retrieved by iFLD
SIF_A_sfm [mW m-2nm-1sr-1] Sun induced fluorescence values for 760 nm retrieved by Spectral Fitting Method (Cogliatti 2015)
SIF_B_sfm [mW m-2nm-1sr-1] Sun induced fluorescence values for 687nm retrieved by Spectral Fitting Method (Cogliatti 2015)
A_sfm_conv [unitless] Convolution coeficient for SFM
B_sfm_conv [unitless] Convolution coeficient for SFM
NDVI [unitless] Vegetation index related to overall greennes , higher values normally mean more greeen biomass (but it saturates at higher levels of biomass)
PRI [unitless] Photochemical reflectance index
MTCI [unitless] MERIS terrestrial chlorophyll index
SR [unitless] Simple ratio
EVI [unitless] Enhanced vegetation index
REP [unitless] Red edge inflexion point
TCARI [unitless] Transformed chlorophyll absorption reflectance index
REDCl [unitless] Red edge crust index
mCRI [unitless] Modified carotenoid reflectance index
EPAR [unitless] Inclomming PAR
LPAR [unitless] Reflected PAR

sat value E full [unitless] Saturation value for incoming radiation at begining of measurement cycle from FLAME
